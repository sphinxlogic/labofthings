Author: Nicholas Kostriken and Robbie G
http://youtu.be/D755eOZ-S24

MentalHouse is a sample app that connects to a Mindwave device via the HomeOS Mindwave driver, and allows you to visualize brainwave signals in the UI
Required device: http://store.neurosky.com/products/mindwave-1
Required binaries: 
	Install the MindWave Mobile Core software that comes on the CD
	The app needs to reference the Mindwave role (RoleMindWave). The source for this custom role is in Hub\Custom\RoleMindWave

To see the code sample for using the Mindwave sensor to interact with a z-wave switch, please see the source file _MentalHouse_withSwitchExample.cs in the same folder
